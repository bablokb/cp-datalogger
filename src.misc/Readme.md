Miscellaneous Sources
=====================

This directory contains various source files for libraries used by the
datalogger.


rtc_ext
-------

This is the source for the rtc_ext module. The compiled version is in
`src.shared/lib9/`.


dataviews
---------

These are the source files from the dataviews-repository
<https://github.com/bablokb/circuitpython-dataviews>. The compiled
version is in `src.shared/lib9/`.

From this project are also the driver files `InkyPack.py` and `phat.py`
for the Pimoroni Inky-Pack and Pimoroni e-ink pHat.
